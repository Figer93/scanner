# CHScanner UI

React + Vite frontend for the lead generation system. Displays the leads table, run panel (start pipeline with limit), and live logs via Socket.IO.

**Run from project root:**

- `npm run dev` – backend + Vite dev server with HMR
- `npm start` – backend only (serves built UI from root `dist/` after `npm run build`)

**From this directory:** `npm run dev` for UI-only dev, `npm run build` to output to `ui/dist/` (root build copies to `../dist`).
